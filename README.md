# laravel-vue-iview
基于laravel+vue以及iview前端ui搭建的后台系统快速开发样板。

> 此项目的搭建过程可以访问[这里](https://segmentfault.com/a/1190000013212484)

## 安装扩展包
```
composer install
```

#### ① 安装npm扩展包
修改来源为国内淘宝镜像
```
npm install -g cnpm --registry=https://registry.npm.taobao.org
```
安装npm扩展包
```
cnpm install
```

#### ② 编译前端代码
```
npm run dev
```

## 访问
```
php artisan serve
```
访问 http://127.0.0.1:8000
